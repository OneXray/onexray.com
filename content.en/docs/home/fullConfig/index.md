# Custom Outbound

"Full Config" requires you to edit the outbound yourself, otherwise there will be no proxy node in the configuration.

Operation path:

Edit Outbounds ➡️ Custom.

Function explanation:

1. Click the "Add" button, you can edit an outbound yourself.
2. Click the "Import" button, you can import an outbound from an existing outbound. Editing imported outbound will not affect the existing outbound.
