点击“+”后会出现二级菜单。

# 输入

## 手动输入

点击选项将会进入 Xray 节点的编辑页面。

## 订阅链接

点击“链接”按钮，将会进入添加订阅页面。

# 导入

所有导入功能均支持以下格式：

1. https 链接。所有 https 链接都被认为是一个订阅链接。
2. VMessAEAD / VLESS。
3. VMessQRCode。
4. Clash.Meta yaml。
5. XrayJson。
6. [OneXray URL Scheme]({{< relref path="../../develop/index.md" lang="zh">}})。

## 扫描二维码

该功能仅适用于 iOS 和 Android 系统。

请注意：当二维码变复杂时，识别率会下降。当分享长链接（如 XHTTP 配置）时，建议您使用文本方式。

## 选择图片

支持常见图片格式，如 png、jpg 等。

## 选择文件

支持图片和文本文件。

1. 图片。支持 png、jpg 和 jpeg。
2. 文件。支持 txt、json 和 yaml。

## 读取剪切板

读取剪切板中的文本数据。
