App 支持您分享以下配置。

1. 所有“Xray 设置”。
2. 所有“节点”。
3. 所有“完整配置”。
4. 所有“原始配置”。
5. 所有“订阅”。
6. 所有“规则集”。

# 通用协议

Xray-core 节点使用 [VMessAEAD / VLESS 分享链接标准](https://github.com/XTLS/Xray-core/discussions/716) 与其他 App 保持兼容。

订阅使用 https 链接格式与其他 App 保持兼容。

# OneXray URL Scheme

OneXray 私有分享协议，主要用于备份和恢复，不同版本间会尽量保持兼容，但推荐您始终使用最新版本生成分享链接。若您为 VPN 服务商，可参考 [开发]({{< relref path="../develop/index.md" lang="zh">}}) 为您的客户提供一键式跳转。

OneXray 私有分享协议支持您分享以下配置。

1. 所有“Xray 设置”。
2. 所有“节点”。
3. 所有“完整配置”。
4. 所有“原始配置”。
5. 所有“订阅”。
6. 所有“规则集”。
