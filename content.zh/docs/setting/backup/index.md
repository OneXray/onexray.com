# 备份

备份功能将会备份您的以下信息。

1. 所有“Xray 设置”。
2. 所有“Local”节点。
3. 所有“完整配置”。
4. 所有“原始配置”。
5. 所有“订阅”。
6. 所有“规则集”。
7. 规则集关联 geosite.dat 和 geoip.dat 文件。

以下信息将不会被备份。

1. 订阅中的“节点”数据。这些数据会在恢复时重新下载。
2. 所有“简易配置”。
3. 其他配置信息。

## 备份文件

备份文件将按日期进行排序。若当日已存在备份，则新的备份文件将覆盖旧的备份文件。

备份文件支持分享和保存。您也可从外部导入备份文件。

# 恢复

恢复时将删除您的所有数据，然后从备份文件中重新导入。

# 系统备份

当您刷机或更换设备时，数据库和配置信息将会被恢复。
但自定义规则集关联的 geosite.dat 和 geoip.dat 文件不会被恢复。
当您恢复系统后，您需要重新下载这些文件。您可点击规则集页面的“刷新”按钮来下载这些文件。

请注意：系统备份将忽略“备份和恢复”生成的备份文件，如 Apple 系统中的 iCloud。您需自行管理这些备份文件。
