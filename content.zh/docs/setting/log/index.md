日志页面包含两个功能。

1. Xray-core 的日志文件。
2. Xray-core 的配置文件。

# 日志文件

当您需要记录 Xray-core 的运行情况时，您可以打开 Xray-core 的日志功能。

操作路径：

1. Xray 设置 ➡️ 自定义设置 ➡️ 编辑 Log。
2. 完整配置 ➡️ Xray 配置 ➡️ 编辑 Log。

您可根据需要调整 `loglevel` 和 打开 `dnsLog`。

# 配置文件

Xray-core 的最终配置文件。当 Xray-core 运行时，内核将读取它们。若您的 VPN 运行不正常，您可在此处检查这些配置文件。
